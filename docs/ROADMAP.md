# Roadmap

## Planned Features

### Phase 1: Enhanced Capabilities (Short-term)

- [ ] **Multi-language Support** - Respond in user's preferred language
- [ ] **Conversation History** - Remember context across sessions
- [ ] **Batch Operations** - Execute multiple related commands
- [ ] **Scheduled Tasks** - Schedule maintenance tasks (cron-like)

### Phase 2: Intelligence (Medium-term)

- [ ] **Learning from Feedback** - Improve based on user corrections
- [ ] **Smart Recommendations** - Suggest optimizations proactively
- [ ] **Security Scanner** - Identify vulnerable packages
- [ ] **Backup Manager** - Automated backup creation/restore

### Phase 3: Integration (Long-term)

- [ ] **Desktop Environment Integration** - KDE Plasma widgets
- [ ] **Web UI** - Browser-based interface
- [ ] **REST API** - Programmatic access
- [ ] **Plugin System** - Extensible architecture

### Phase 4: Advanced Features

- [ ] **Voice Commands** - Speech input/output
- [ ] **Multi-agent Collaboration** - Specialized sub-agents
- [ ] **Container Management** - Docker/Podman integration
- [ ] **Virtual Machine Control** - VM lifecycle management

## Known Issues

- Some performance tuning features require persistent sudo (consider policykit)
- Web search requires `ddgr` to be installed
- Limited to apt-based distributions (extensible to others)
